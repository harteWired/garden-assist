# Garden Assist — Warren County NJ 2026

Home vegetable garden plan for two 4x12 raised beds (12" depth) in Warren County, NJ (Zone 6b). Part shade conditions (~5-6 hours midday sun). Last frost ~May 10-15.

## Quick Start

Open `app/garden-hq.html` in any browser — it's a self-contained single-page app with:
- **Bed Layout** — interactive grid showing plant placement for both beds
- **Timeline** — planting schedule from seed start through fall
- **Shopping Checklist** — persistent checkboxes (saved in localStorage)
- **Fence Build** — groundhog exclusion fence instructions

## Project Structure

```
app/                  Interactive HTML planner (primary deliverable)
planning/             Garden plan details, shopping spreadsheet
guides/               Build guides (groundhog fencing)
docs/                 Photos, measurements, yield tracking (future)
Input-drafts/         Original source files (reference only)
```

## Key Dates

| When | Action |
|------|--------|
| Mar 14-16 | Start brassicas + lettuce indoors (soil blocks) |
| Mar 28-Apr 1 | Start tomatoes, chard, parsley indoors |
| Apr 1-15 | Build groundhog fence |
| Apr 10-20 | Direct sow cool-season crops outdoors |
| May 20-25 | Transplant tomatoes + basil |

## Site Conditions

- **Zone 6b**, last frost ~May 12
- **Part shade**: ~5-6 hrs direct midday sun (evergreen + house shade)
